Portflio Projects

KeepIt

KeepIt is an iOS App built to help users keep track of all their personal belongings without the use of gps hardware. It does this through customizable notifications that can be timed or triggered upon crossing the border of a personalized GeoFence. Whenever you lose an item, you can use KeepIt to remember where you had it last.

Demo Repository

Maze Solver

Maze Solver is a python project which seeks to test the efficiency of many known maze solving algrithms. It starts by generating a completely random maze and then attempts to solve it using the left-hand-rule, dead end elimination, and depth first search algorithms. The solving times for each of these algorithms are then compared.

Demo Repository
